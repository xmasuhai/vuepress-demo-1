## 模块概述

- 技术文档 或 个人博客
- 静态网站：不和后端交互，没有从数据库那数据，直接把数据写在前端
- 用Markdown来写，生成HTML网站
  - 生成目录结构到左侧 SideBar 中
  - 内部锚点可重定向
  - 内置搜索关键字
- Markdown 中可以插入 Vue 全局组件
